# MyRental — Website Company Profile Rental Mobil

Website profil perusahaan rental mobil premium lengkap dengan panel admin untuk mengelola data.

---

## Halaman Utama (Website)

Yang dilihat pengunjung website.

### 1. Navigasi Atas (Navbar)

Menu tetap di bagian atas halaman. Berisi logo MyRental, link ke bagian-bagian website, dan tombol "Hubungi Kami". Saat di-scroll, latar belakangnya berubah jadi putih. Di handphone, menu berubah jadi tombol hamburger yang bisa diklik.

Link navigasi:
- **Beranda** — ke bagian paling atas
- **Paket** — ke daftar paket rental
- **Katalog** — ke daftar mobil
- **Testimoni** — ke ulasan pelanggan
- **Kontak** — ke bagian footer

### 2. Hero (Pembuka Halaman)

Tampilan full-screen pertama yang dilihat pengunjung. Ada foto latar kota gedung pencakar langit, tulisan besar "Solusi Rental Mobil Premium untuk Perjalanan Anda", dan dua tombol: "Lihat Katalog" dan "Hubungi Kami". Ada juga tiga badge kepercayaan: Armada Terawat, Booking Cepat, Harga Transparan.

### 3. Paket Rental (Pricing)

Tiga pilihan paket sewa mobil dalam bentuk kartu:
- **Lepas Kunci** (Rp 350.000/hari) — sewa mobil tanpa supir
- **Include Supir** (Rp 500.000/hari) — sewa + supir, ini yang paling populer (ada badge "POPULER")
- **Include Supir + BBM** (Rp 650.000/hari) — sewa + supir + bahan bakar full

Setiap kartu menampilkan fitur-fitur yang didapat dan tombol "Pilih Paket".

### 4. Keunggulan Layanan

Enam ikon keunggulan yang menunjukkan nilai lebih MyRental:
- Armada Terawat
- Driver Profesional
- Harga Transparan
- Booking Cepat
- Support 24 Jam
- Perjalanan Aman

### 5. Grid Fitur Unggulan (Bento Grid)

Tujuh kartu dengan ukuran berbeda yang menonjolkan fitur layanan:
- Premium Armada (kartu merah besar)
- 24/7 Support
- Fast Booking
- Full Insurance (kartu gelap)
- Flexible Rental
- Safe Journey
- Pro Driver

### 6. Katalog Mobil

Enam mobil pilihan dengan foto asli:
- Toyota Avanza (7 seats, Manual)
- Toyota Innova (7 seats, Manual/Matic)
- Honda Brio (5 seats, Manual/Matic)
- Mitsubishi Pajero (7 seats, Matic)
- Toyota Hiace (12 seats, Manual)
- Toyota Fortuner (7 seats, Matic)

Setiap kartu menampilkan gambar mobil, nama, kategori, jumlah kursi, transmisi, harga per hari, dan tombol "Sewa". Di handphone, daftar mobil bisa digeser ke kiri-kanan (carousel).

### 7. Statistik Perusahaan

Angka-angka untuk membangun kepercayaan: 50+ Armada, 500+ Pelanggan Puas, 5+ Tahun Pengalaman, 10+ Kota Cakupan. Dengan latar merah.

### 8. Testimoni Pelanggan

Tiga ulasan dari pelanggan asli dengan foto, nama, pekerjaan, dan rating bintang 5.

### 9. CTA Banner (Ajakan Bertindak)

Banner merah dengan ajakan "Siap untuk Perjalanan Terbaik Anda?" dan dua tombol: "Hubungi via WhatsApp" dan "Telepon Sekarang".

### 10. Footer (Kaki Halaman)

Informasi lengkap di bagian bawah:
- Logo dan deskripsi perusahaan
- Link menu dan layanan
- Informasi kontak (telepon, email, alamat)
- Ikon media sosial (Instagram, Facebook, Youtube)
- Copyright

## Panel Admin

Halaman khusus untuk mengelola data website. Hanya bisa diakses setelah login.

### Login

Halaman masuk dengan foto mobil premium di latar belakang. Cukup masukkan username dan password, lalu klik "Masuk".

### Dashboard (Beranda Admin)

Tampilan utama setelah login. Menampilkan:
- **4 kartu statistik**: Total Armada, Paket Rental, Testimoni, dan Pending Order
- **Grafik batang** jumlah order per bulan
- **Progress bar** status order (Pending, Confirmed, Completed)
- **Tabel** 5 order terbaru

### Data Order (Manajemen Pesanan)

Fitur untuk mencatat dan mengelola pesanan rental:
- **Melihat daftar** semua order dalam bentuk tabel
- **Menambah order baru** dengan mengisi form: nama pelanggan, telepon, paket, mobil, tanggal mulai, durasi
- **Mengubah status** order: Pending (menunggu), Confirmed (dikonfirmasi), Completed (selesai), Cancelled (dibatalkan)
- **Menghapus** order jika diperlukan

### Konten Website (Manajemen Konten)

Fitur untuk mengubah konten yang tampil di website:

**Atur Data Mobil:**
- Melihat daftar mobil yang tampil di katalog
- Mengedit nama, kategori, harga, dan spesifikasi mobil
- Menambah mobil baru
- Menghapus mobil
- Reset ke data awal jika ada kesalahan

**Lihat Data Paket:**
- Menampilkan paket rental yang sudah ada

**Lihat Data Testimoni:**
- Menampilkan testimoni yang sudah ada

### Navigasi Panel Admin

Di sisi kiri ada menu untuk berpindah halaman:
- Dashboard (beranda admin)
- Orders (data pesanan)
- Konten (atur konten website)
- Ke Website (kembali ke halaman utama)
- Logout (keluar dari panel admin)

## Alur Penggunaan

### Untuk Pengunjung Biasa:
1. Buka website
2. Lihat hero section dan scroll ke bawah
3. Baca paket rental, keunggulan, fitur
4. Lihat katalog mobil
5. Baca testimoni
6. Klik tombol WhatsApp atau telepon untuk order

### Untuk Admin:
1. Buka halaman `/login`
2. Masuk dengan username dan password
3. Cek dashboard untuk melihat statistik
4. Kelola order masuk (ubah status, hapus)
5. Edit data mobil di menu Konten jika perlu
6. Klik Logout setelah selesai
