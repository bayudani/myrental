# AGENTS.md

## Project Overview

Project name: MyRental
Type: Company Profile / Landing Page
Stack: React + TanStack Start + TailwindCSS + Shadcn UI
Nature: Static Website (No backend, no database)

Goal:
Build a premium and modern car rental company profile website with a clean, professional, high-conversion design.

IMPORTANT:
This project must strictly follow the provided design reference.

Core concept:

- Modern automotive branding
- Premium feel
- Clean white base
- Strong red accent color
- Big typography
- Strong CTA
- Smooth section flow

---

## Main Sections

Required sections:

1. Navbar
2. Hero Section
3. Rental Packages
4. Service Advantages
5. Bento Grid Features
6. Car Catalog
7. Statistics Counter
8. Testimonials
9. CTA Banner
10. Footer

---

## Rules

- DO NOT redesign freely
- Follow spacing consistency
- Keep visual hierarchy strong
- Focus on conversion
- Mobile responsive required
- Use smooth scroll navigation
- Use reusable components

## UI Style

Style should feel:

- Premium
- Fast
- Automotive
- Corporate
- Clean
- Minimal but rich

Avoid:

- Too playful
- Too colorful
- Over-rounded elements
- Glassmorphism
- Neon effects

## Components Strategy

All sections must be separated into:
src/components/sections/

src/components/ui/

Data:
src/data/

src/assets/
